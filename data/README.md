# Data

All datasets are downloaded automatically via HuggingFace `datasets` at runtime — no manual downloads needed.

**Datasets used:**
- **SST-2** (Stanford Sentiment Treebank) — binary sentiment classification, ~67K train examples
- **MRPC** (Microsoft Research Paraphrase Corpus) — paraphrase detection, ~3.7K train examples
- **WikiText-2** — language modeling benchmark, ~2M tokens
